# Telegram Shop Bot

Телеграм-бот для продажи аккаунтов, валюты и недвижимости с встроенной системой тикетов и админ-панелью.

## Возможности

- 🛒 Магазин аккаунтов с автовыдачей логина/пароля после подтверждения оплаты
- 💼 Заявки на продажу аккаунта
- 💰 Покупка и продажа игровой валюты
- 🏠 Заявки по недвижимости
- ⭐ Отзывы
- 🆘 Поддержка с тикетами и ответами админа
- ⚙️ Админ-панель: добавление/удаление аккаунтов, управление тикетами

## Установка

```bash
git clone https://github.com/<your-username>/<repo>.git
cd <repo>
pip install -r requirements.txt
```

## Настройка

Скопируйте `.env.example` в `.env` и заполните своими значениями:

```bash
cp .env.example .env
```

- `TOKEN` — токен бота от [@BotFather](https://t.me/BotFather)
- `GROUP_ID` — ID админской группы (бот должен быть в неё добавлен)

Либо просто экспортируйте переменные окружения:

```bash
export TOKEN="123456:ABC..."
export GROUP_ID="-1001234567890"
```

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — открыть меню пользователя
- `/panel` — открыть админ-панель (только в админ-группе)

## Структура

- `bot.py` — основной файл бота
- `bot.db` — локальная SQLite база (создаётся автоматически)
- `requirements.txt` — зависимости
